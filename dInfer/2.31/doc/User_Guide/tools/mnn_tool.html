MNN原生工具

使用说明

MNN官方说明：MNN Docs - 测试工具

必要的binary程序以及python脚本已经编译并打包到dInfer中，使用原生工具前，请将${dInferSDK}/${PLATFORM}/bin/路径加入到环境变量

对于Windows PowerShell用户，请使用以下命令：

$env:PATH += ";${dInferSDK}/${PLATFORM}/bin"

对于Linux用户，请使用以下命令：

export PATH="${dInferSDK}/${PLATFORM}/bin:$PATH"

建议步骤

  • 使用MNN Docs - 模型转换中的testMNNFromOnnx.py脚本，一键进行模型转换&CPU侧的正确性验证，正确性验证出错时通过MNN Docs - 模型转换 - 出错及解决进一步分析

  • 使用ModuleBasic工具中的ModuleBasic.out工具，进行特定后端配置的性能评估，例如CUDA后端、不同精度、内存、功耗模式等

  • 若希望进一步验证一致性，可使用checkFile.out工具和checkDir.out工具检查生成的txt文件，也可以将txt文件通过np.loadtxt()读取通过python进一步处理。

  • 若希望进一步分析算子耗时，可设置ModuleBasic工具的runMask参数为4，以打印算子耗时。对于CPU后端，也可以使用timeProfile工具
